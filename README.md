# GreyAI
This is a proof of concept of a AI in greyhack, People are free to edit and clone this repo.

## Features
- Emotional Simulation
- Mood Management
- Long-Term Memory
- Self-Reflection
- Adaptable Learning Rate
- Response Complexity
- Interaction Depth
- Formal vs. Casual Response Styles
- Feedback Response
- Dynamic Responses
- Exit Command
